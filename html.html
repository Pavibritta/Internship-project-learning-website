HTML Course

Learn the foundation of building websites with HTML.

1. Introduction to HTML

HTML (HyperText Markup Language) is used to create the structure of a website. 

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

2. Structure of a Web Page

  • <html> – root element
  • <head> – metadata, title, links
  • <body> – visible content

3. Headings, Paragraphs, Lists

<h1>Main Heading</h1>
<p>This is a paragraph.</p>
<ul>
  <li>Item One</li>
  <li>Item Two</li>
</ul>

4. Links & Images

<a href="https://google.com">Go to Google</a>
<img src="image.jpg" alt="Sample Image">

5. Forms & Inputs

<form>
  <input type="text" placeholder="Your Name">
  <input type="email" placeholder="Email">
  <button>Submit</button>
</form>

6. Final Project

You will build a complete webpage with header, content sections, and footer.
